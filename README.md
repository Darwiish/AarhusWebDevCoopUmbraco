Introduction

This project is the first mandatory assignment for the Development Environment course. The goal was to create a web application using:

Microsoft Visual Studio (IDE)

ASP.NET Core MVC Framework

Umbraco CMS

MS SQL Server

The website is designed for web developers to communicate with the public and customers, manage projects, and organize collaboration.

The project includes:

A functional web application

Documentation of development processes

GitHub repository with full source code

Sitemap

A Partial View was created to generate a sitemap.

A Sitemap Document Type was created based on the Master template.

The sitemap is dynamically displayed as part of the main menu.

Website Menu

Menu generated by iterating through visible child pages in the site model.

Dropdown items are dynamically filtered and ordered.

Sub-items are checked for visibility and access rights before rendering.

Website Access Control
Users

Two users were created in Umbraco Backoffice:

Jane Wilfred – Editor, full access

Tim Hansson – Writer, access limited to News section

Members

Three member groups: Active, Guest, Public

Different member types created with specific access rights.

Login system implemented with partial views and surface controllers.

Restricted pages redirect to login for unauthorized users.

Contact Form

Implemented with a Surface Controller and server-side/client-side validation.

Submissions create a new Umbraco Document and optionally send emails using SMTP.

Validations implemented via data annotations in the view model.

Conclusion

This project provided hands-on experience with Umbraco CMS, ASP.NET Core MVC, and web application security/access control. Key takeaways include:

Structured website navigation with sitemaps and menus

Role-based access control for users and members

Integration of contact forms with server-side validation and email support

Logging and content management via Umbraco
